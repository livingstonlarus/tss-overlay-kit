# Tech Stack (DE-002 Standard)

- **Frontend**: SolidJS (Fine-grained reactivity)
- **Meta-Framework**: TanStack Start (Solid + Vinxi)
- **Styling**: Tailwind CSS + PostCSS (NO `@tailwindcss/vite`)
- **UI Library**: Kobalte (Headless Primitives)
- **Database**: PostgreSQL (Neon Serverless via standard `pg` driver)
- **ORM**: Drizzle ORM
- **Authentication**: Better Auth (User Override - confirmed v4.1 standard for this user)
- **Process Management**: PM2
- **Validation**: Zod / Valibot
- **Email**: Resend
- **Tooling**: @tanstack/cli (Scaffolding & MCP)
